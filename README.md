# 🎓 Student Placement Prediction using Machine Learning

## 📌 Project Overview

This project predicts whether a student is **Placed** or **Not Placed** using a Machine Learning classification model. The model is trained on student academic and performance data using the **Random Forest Classifier** algorithm.

---

## 🚀 Features

- Load student performance dataset
- Preprocess the data
- Train a Random Forest Classifier
- Evaluate model accuracy
- Accept user input for a new student
- Predict the student's placement status

---

## 🛠️ Technologies Used

- Python
- Jupyter Notebook
- Pandas
- Scikit-learn

---

## 📂 Dataset Features

The model uses the following features:

- Study Hours
- Attendance
- Sleep Hours
- Internet Usage
- Assignments Completed
- Previous Score
- Exam Score

### Target Variable

- Placement Status

---

## 🤖 Machine Learning Algorithm

- Random Forest Classifier

---

## 📊 Model Evaluation

The model is evaluated using:

- Accuracy Score

---

## ▶️ How to Run

1. Clone this repository.

2. Install the required libraries.

```bash
pip install -r requirements.txt
```

3. Open the Jupyter Notebook.

4. Run all cells.

5. Enter the student's details when prompted.

6. The model will predict the student's placement status.

---

## 💻 Example Input

```
Study Hours: 8
Attendance (%): 90
Sleep Hours: 7
Internet Usage (hours): 3
Assignments Completed: 18
Previous Score: 85
Exam Score: 92
```
### Example Output
```
Predicted Placement Status: Placed
```
